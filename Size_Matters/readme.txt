# Size Matters

A clothing size recommendation system that helps users find their perfect fit without requiring measurements.

## Overview

Size Matters is a user-friendly application that recommends clothing sizes based on body type rather than numerical measurements. Users simply select the body type that most closely resembles their own, and the system provides personalized size and style recommendations for various clothing items.

## Features

- Visual body type selection (no measurements needed)
- Size recommendations for multiple clothing categories
- Style tips tailored to each body type
- Simple, intuitive user interface
